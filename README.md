# Classyco Growth OS

**Módulo 1:** Painel de Diagnóstico de Growth Marketing para o mercado imobiliário.

## 📋 Conteúdo

- **PRD** (`docs/PRD.md`) - Especificação completa do produto
- **Briefing** (`docs/BRIEFING.md`) - Visão geral do projeto
- **Protótipo** (`reference/prototype/dashboard-prototype-v3.jsx`) - Componente React do dashboard

## 🚀 Sobre

Sistema de diagnóstico baseado no método "Destrava Receita" para identificar gargalos em growth marketing. Focado em gestão de tráfego pago e canais de aquisição.

**MVP:** Próximo para Rui da Cruz (corretor autónomo em Portugal)

## 📦 Stack

- **Next.js 14+** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Notion API** (Data Hub)
- **Claude API** (geração de relatórios)
- **Vercel** (hosting)
